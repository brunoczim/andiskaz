pub fn keycodes_per_modifier(&self) -> u8

Get the value of the keycodes_per_modifier field.

The keycodes_per_modifier field is used as the length field of the keycodes field. This function computes the field’s value again based on the length of the list.

Panics

Panics if the value cannot be represented in the target type. This cannot happen with values of the struct received from the X11 server.
