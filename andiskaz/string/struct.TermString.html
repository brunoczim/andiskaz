pub fn new<S>(string: S) -> Result<Self, TermStringError> where
    S: AsRef<str>, 

Builds a new graphical string.

The string must not start with a diacritic character. Diacritic here is not “^” or “~”, but rather a diacritic that when inserted combines with the previous character. Like the tilde in “ỹ” which can be separated from “y”. On the other hand, the combination “ỹ” is valid and forms a single grapheme cluster. The diacritic is only invalid when separated.

Control characters also trigger an error, because those would allow the terminal to be controlled.
